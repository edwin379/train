# 都営 TOEI LIVE

Real-time Tokyo Toei transit map.

## Requirements
- [Node.js v18+](https://nodejs.org/)

## How to Run

1. Install dependencies
```
npm install
```

2. Start the server
```
npm start
```

3. Open your browser and go to
```
http://localhost:3000
```

## How to Stop
Press `Ctrl + C` in the terminal.
